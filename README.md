# GDS Test

NVIDIA GPUDirect Storage (GDS) test code using [Cufile API](https://docs.nvidia.com/gpudirect-storage/api-reference-guide/index.html)

## Build and run

```bash
# Build
make
```
```bash
# Run
./gds [device path] # e.g., ./gds /dev/nvme0n1
```
